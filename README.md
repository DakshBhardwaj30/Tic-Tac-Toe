# Kivy Tic Tac Toe 🎮

A beautiful Tic Tac Toe game built with Python and Kivy. Play against your friend or challenge the AI bot!

## Features
- Human vs Human & Human vs Bot modes
- Intuitive UI with emojis for ❌ and ⭕
- Play Again and Restart options

## Run it
```bash
python main.py
